# Roadmap Archive

This folder is the detailed implementation archive for DesignMind.

## What Lives Here
- `phase-*.md`: one roadmap per implementation phase
- `commit-*.md`: one roadmap per meaningful historical commit
- `master-roadmap.md`: index of the archive

## Status Labels
- `Finalized`: completed artifact aligned to the shipped repo
- `Reconstructed`: backfilled from git history, changelog, and current repo state
- `Planned`: reserved for future work that has not shipped yet

## Naming Rules
- Phase files use `phase-0N-<topic>.md`
- Commit files use `commit-<hash>-<topic>.md`
- Files should reference real paths, real commands, and real commit hashes only

## Canonical Relationship
- `aiDocs/` remains the canonical reviewer-facing narrative
- `ai/roadmaps/` is the detailed supporting evidence archive
